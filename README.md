# Color themes for Emacs that was stolen from John Blow live streams

![jblow-nostalgia](./jblow-screenshot.png)

## Instalation
1. Clone the repository to your computer
2. Add following line into you emacs config (.emacs or .emacs.d/init.el)
```emacs-lisp
(add-to-list 'custom-theme-load-path "~/your/path/to/jblow-nostalgia/")
```
3. Call `customize-theme` extension function and select jblow-nostalgia theme.

That it.
